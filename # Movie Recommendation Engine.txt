# Movie Recommendation Engine

This project demonstrates how to build a movie recommendation engine using PySpark's MLlib.

## Features
- Load and preprocess movie and rating data
- Train a recommendation model with ALS
- Evaluate model performance
- Generate movie recommendations for users

## Setup
1. Install PySpark: `pip install pyspark`
2. Clone the repository: `git clone <repo-url>`
3. Run scripts from the `src` directory.

## Dataset
Use the MovieLens dataset (sample provided in the `data` folder).
